### Make Predictions
```python
from app.predict import predict_heart_disease

patient = {
    'age': 63.0, 'sex': 1, 'cp': 1,
    'trestbps': 145.0, 'chol': 233.0,
    'fbs': 1, 'restecg': 2, 'thalach': 150.0,
    'exang': 0, 'oldpeak': 2.3,
    'slope': 3, 'ca': 0.0, 'thal': 6.0

```
